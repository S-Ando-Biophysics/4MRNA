Programming Code Generator for 4MRNA

Step 1

Create a new directory on your own computer for running 4MRNA.

After creating it, place your reflection file (.mtz) for molecular replacement into that directory.

Step 2

Fill in the table below and click "Generate codes".

For "Working directory", enter the path of the directory you created in Step 1. (This field can be left blank when using Basic Version.)

When you select "No. of models", the corresponding number of input rows will appear below. Then enter each model’s information there.

"No. of AU" indicates how many copies of the model to be searched in molecular replacement.
